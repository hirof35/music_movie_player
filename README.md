🎬 Pro-Spec Multi-Media Player
~ Pythonによる高度な動画・音声同期エンジニアリング ~
Pro-Spec Multi-Media Player は、モダンなデザイン（CustomTkinter）と、プロ仕様のメディア制御ライブラリを組み合わせた、高機能なマルチメディアプレイヤーです。

🌟 概要
一般的なGUIアプリでは困難な「動画フレームと音声の厳密な同期」を、ffpyplayer のクロック同期ロジックを利用して解決しています。MP4動画だけでなく、MP3/WAVといった音楽ファイルにも対応し、シークバーによる直感的な操作が可能な「オールインワン」の再生環境を提供します。

✨ 特徴機能
高度な動画・音声同期 (A/V Sync):

ffpyplayer の MediaPlayer をエンジンに採用。動画のコマ送りと音声の再生速度をミリ秒単位で同期させます。

モダンなダークモードUI:

CustomTkinter を全面的に採用。OSのテーマに合わせた洗練されたデザイン。

ダイナミック・スケーリング:

ウィンドウサイズに合わせて動画をリアルタイムでリサイズ。アスペクト比を維持しながら最適な視聴体験を提供します。

統合型プレイリスト:

動画と音楽を混在させて管理。ダブルクリックでの即時再生、個別削除、全削除に対応。

フル・コントロール:

高精度なシーク操作、音量調整、再生/一時停止、ステータス表示機能を完備。

🛠 技術スタック
GUI: CustomTkinter (Tkinterのモダン拡張)

Video Engine: OpenCV (フレーム抽出) / ffpyplayer (音声同期・デコード)

Audio Engine: pygame.mixer (音楽ファイル再生)

Image Processing: Pillow (フレーム変換)

🚀 インストールと実行
1. 依存ライブラリのインストール
Bash
pip install customtkinter opencv-python pillow pygame ffpyplayer
2. 実行
Bash
python media_player.py
🕹 使い方
メディア追加: 「ファイル追加」から、動画（.mp4）や音楽（.mp3, .wav）を選択します。

プレイリスト: 右側のリストをダブルクリックすることで、再生対象を切り替えられます。

シーク: スライダーを動かすことで、動画や音楽の任意の場所へ瞬時に移動できます。

音量: 下部の音量スライダーで、動画・音楽共通の音量調整が可能です。
